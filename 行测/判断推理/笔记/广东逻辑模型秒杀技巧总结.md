# 广东逻辑模型秒杀技巧总结

说明：本总结基于《（已更新）宝藏级资料！广东逻辑模型秒杀技巧.pdf》的内容，将 15 类常见逻辑题型（包含形式推理与真假命题类）按题型特点、解题技巧与快速步骤归纳，便于快速复习与实战使用。所有技巧为有限验证的经验型方法，使用时务必结合题干验证。

---

## 总体答题注意事项（通用）
- 先把前提句子逐条列清楚，标注“有些 / 所有 / 如果…则 / 或 / 并且 / 否定”等关键词。
- 秒杀技巧是有限性总结，遇到生僻组合先用传统推理或必要验证复核。
- 解题时重点：找“唯一真/唯一假/被点名/共同主体/数量对比”等关键信息。
- 做题流程建议：读题 → 摘要前提 → 识别模型 → 套用秒杀法 → 验证选项。

# 第一部分：形式推理的秒杀法（1–9）

1. 至少共同法（“至少1/至少2/至多1”类）
- 题型特点：
  - 出现“至少1种/至少2种/至多1种”等数量句；
  - 常有复合假言限制（若A则非B）。
- 解题技巧（快速三步）：
  1. 找到“至少1”的句子（候选集合A）。
  2. 找到“至少2”的句子（集合B）。
  3. 两句的共同主体即为必然项（或排除法得到必然项）。
- 注意：遇到“至多”句要结合其它“至少”句做选择性验证。

2. 高低主体搭桥法（数量比较 + 两个范畴）
- 题型特点：
  - 涉及“高于/低于/更多/更少”等比较；
  - 两个范畴交叉（如工作表现 vs 帮助关系）。
- 解题技巧：
  1. 找出高低主体（谁高谁低）。
  2. 找出两组范畴（如帮助关系 vs 税额）。
  3. 按顺序“搭桥”：从高往低选“少的”，从低往高选“多的”。
- 常见陷阱：不要把传递性质随意反向传播，确认中间环节角色。

3. 大大则大二分法（“大于50%”类）
- 题型特点：
  - 两个互斥集合划分（例如政治面貌只有A和B两类）且存在“某类占多数”。
- 解题技巧：
  - 利用占多数 (>50%) 的交叉对比，通常能推出“多数主体与另一组交叉多数/占优”的结论。
- 验证：注意“多数”与“全部/一些”的区别，谨防把“多于”误写为“全部”。

4. 不同中找相同 — 奇数类（5 人类、3:2 分布）
- 题型特点：
  - 有 5 个主体，属性分布常为 3 vs 2，出现“甲乙相同/丙丁不同/只有一人满足”的信息。
- 秒杀技巧：
  1. 已知“甲乙相同”，可直接排除甲乙作为唯一满足者；
  2. 在剩余的不同组合里按“不同中找相同”进行定位。
- 常见套路：用“相同排除法”快速剔除选项。

5. 不同中找相同 — 偶数类（4 人类）
- 题型特点：
  - 四位候选，属性分布如 3 满足 A、2 满足 B，且只有 1 人满足 A∧B∧C。
- 秒杀技巧：
  - 先找“不满足某条件的人”（通常通过并非句），该人对应的“队友”即可能是答案（队友：属性相同者）。
- 验证点：注意题干是否说明“每人至少满足一项”等限制。

6. 欺软怕硬（“所有 / 有的 / 点名/顺着/反着”）
- 题型特点：
  - 常有“所有 X 是 Y、某些 Z 是 …”等全称和特称混合；
  - 存在点名/特殊句或一部分顺着、一部分反着陈述。
- 秒杀技巧：
  - 找点名句的客体与“所有”句的方向；把“所有顺着讲、有的反着讲”作为破题线索。
- 注意：不要把“所有”错误外推为“唯一”，谨慎验证。

7. 同岗收队法（假言 + 三主体）
- 题型特点：
  - 三个主体，多为“如果甲和乙都不…，则丙必…”类型。
- 秒杀技巧（思路）：
  - 若题目要求推出“甲选了”，则找到使得“甲必须为真”的附加前提（通常是让其它两个都不成立的设定）。
- 备注：题型较少见，遇到先验证假言前提。

8. 顺推主体相消法（“如果…那么…/或”转化）
- 题型特点：
  - 多为翻译推理，含“如果…则…”、“或”命题。
- 秒杀技巧：
  1. 把“或”命题合理转化为条件链（例如非A → B）；
  2. 消除不可能成立的主体，剩下的即为答案。
- 关键在于正确把“或”与“非”互换并消去不合条件的主体。

9. 三段秒杀法（三段论快速判）
- 题型特点：
  - A→B，B→C，推 A→C 的经典结构，可能存在全称/特称、肯定/否定的混合。
- 秒杀技巧要点：
  1. 注意中项（B）是否“周延”；
  2. 两肯定前提得出肯定结论；肯定+否定常得否定结论；
  3. 若前提含特称（有些），结论通常为特称；两个否定前提不能得有效结论。
- 常见快速检查清单：是否有全称/特称？是否有否定？结论方向是否被允许由大推小或反向推理？

# 第二部分：真假命题的秒杀法（10–15）

10. 或然为真（四句两真两假模型）
- 题型特点：
  - 大前提四个陈述、其中两真两假；小条件常含“所有非 / 有的是 / A 是/ A 或 B 是”。
- 秒杀技巧：
  - 识别“或然必真”：通过排除使“所有/并且”为假的情况，保留“有的/或则为真”的结论。
- 要点：把“有的真、所有假、并且假”作为判断核心。

11. 绝对大多为假（四句两真两假，含绝对命题）
- 题型特点：
  - 四个陈述两真两假，常含“绝对”或强语（所有/并且）句子。
- 秒杀技巧：
  - 把可被判为“绝对假”的项找出，并据此推断剩余个体的真值分布。
- 验证：分析哪两句可同时为真，哪两句必为假，推断结果。

12. 最大信息法（多次猜测/多次判断）
- 题型特点：
  - 多次猜测或陈述，存在“全对/半对/全错”或“对2次/对1次/全错”等限制。
- 秒杀技巧：
  - 找出出现频率最高（出现三次或两次且一致）的主体或属性，常为答案（最大信息原则）。
- 示例思路：若某属性在多个陈述中重复出现且满足“只一个人两次都对”，则对应项为正确。

13. 最小信息法（3x3 交叉搭配）
- 题型特点：
  - 三个主体、三种属性的配对猜测，且每个猜测“只对一半”。
- 秒杀技巧：
  - 找最小信息（那条仅给出一半真值的信息），用交叉验证排除错误搭配，最终代入剩余匹配。
- 注意：这是有限验证型，需要把每种组合代入验证一致性。

14. 上顺下反（上下反对法）
- 题型特点：
  - 上位句（通常为“所有是/所有非”）与下位句（“有些是/有些非”）构成真假反对情形；
  - 适用于“只有一真/只有一假”的题型。
- 秒杀技巧（上顺下反处理）：
  1. 忽略“反对”先判断常规含义；
  2. 找出“落单”命题（与其他两句矛盾的那句）；
  3. 将主体视为全称，按上反/下反规则把“没有”或“有”反转成结论。
- 常见类型：上下层次混合真假，需要注意“只有一真/只有一假”的约束。

15. 点名秒杀法（点名某句为真）
- 题型特点：
  - 小题中某条被点名为“真”或“假”，并且存在“只有 A 为真/我不是 A 但只有 A 为真”等设定。
- 秒杀技巧：
  1. 找出被“点名”的句子；
  2. 根据点名句确定对应主体或属性的真假；
  3. 点名句与被点名对象通常都为假/真（资料中常用“点名和被点名都假”的总规则，具体题型要核验证据）。
- 典型应用：四人说真话类题，确定被点名者真假后快速排除并定位身份。

# 快速答题检查表（考场小抄）
- 有“至少、至多、有些、所有”字样？→ 使用“至少共同法 / 大小法 / 三段论”。
- 出现“如果…则…/或”结构？→ 优先考虑“顺推主体相消法 / 或转条件”。
- 有“4句两真两假/只有一真/点名句”？→ 用真假命题模型（或然/绝对/点名/上顺下反）。
- 出现“3x3 配对/两次猜测/频次提示”？→ 用“最小信息法 / 最大信息法”。
- 出现数量比较（多于/少于）且集合划分明确？→ 用“大大则大二分法”。
- 看到“甲乙相同/丙丁不同/只有一人满足”？→ 尝试“不同中找相同（奇/偶类）”的排除法。

---

# 常见误区与防护
- 把“多于”误当作“全部”或“必须” → 时刻区分“多数/全部/有些/有的”。
- 直接外推中项（B）特性到非周延情形 → 三段论时检查中项是否周延。
- 忽视题干限定（例如“每人至少符合一项”） → 先确认题干的全局限制。
- 过度依赖单一秒杀公式，不做选项验证 → 秒杀后要做简短验证（常为 1–2 步）。

---

# 练习建议
- 针对每种模型至少做 10 道题，优先练习广东/省考真题中相同题型。
- 做题时严格写出每一步验证（尤其真假命题类），培养直觉同时保持准确。
- 建立“模型笔记”：记录每类题型的典型前提组合与反例，方便考前速记。
